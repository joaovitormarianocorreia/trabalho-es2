# Dashboard COVID-19 

Projeto Final da disciplina Engenharia de Software II, da Universidade Estadual Paulista, campus de Bauru. 

## Terminal Commands

1. Instale o NodeJs através [NodeJs Official Page](https://nodejs.org/en).
2. Abra o terminal
3. Navegue até a pasta do projeto
4. No terminal, rode o comando: ```npm install -g @angular/cli```
5. Depois: ```npm install```
6. E ainda: ```ng serve```
7. No navegador acesse `http://localhost:4200/`
